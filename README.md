# VitaAI

VitaAI 是一个面向普通用户的 AI 驱动健康管理系统，聚焦体检解读、风险提醒和器官专题管理。

它不试图替代医生，也不直接给出医疗诊断结论，而是帮助用户更早发现潜在风险，理解身体指标变化，并围绕不同器官系统获得更清晰、可执行的生活方式建议。

## 产品定位

现代人每年都会做体检，但很多报告看不懂、异常项不知道是否严重、复查和干预也缺少持续管理。VitaAI 的目标，是把一次性的体检结果变成持续性的健康管理过程。

VitaAI 面向以下用户：

- 看不懂体检报告，想知道异常指标意味着什么的人
- 有慢病风险、家族史或长期亚健康状态的人
- 想围绕肝脏、心血管、肠胃、肺部等重点器官做长期预防管理的人
- 希望获得更具体生活方式建议，而不是泛泛“多喝水、多运动”的人

## 核心能力

### 1. AI 体检报告解读

系统支持对体检报告、化验单和关键健康指标进行结构化分析，帮助用户理解：

- 哪些指标异常
- 异常可能关联到哪些器官系统
- 哪些问题需要尽快复查
- 哪些情况更适合先观察和生活方式调整
- 指标之间是否存在联动风险

例如：

- 转氨酶、脂肪肝、血脂异常可关联肝脏与代谢风险
- 血压、心率、血脂、血糖可关联心血管风险
- 胃肠症状、便检、幽门螺杆菌结果可关联消化系统管理
- 肺结节、吸烟史、呼吸症状可关联肺部专项关注

### 2. 器官专题健康管理

VitaAI 不是只看单个指标，而是围绕重点器官建立专题管理能力。

首期可覆盖的方向包括：

- 肝脏管理：脂肪肝、转氨酶异常、肝功能波动、饮酒相关风险
- 心血管管理：高血压、高血脂、动脉硬化倾向、胸闷心悸风险提醒
- 肠胃管理：胃炎、反酸、幽门螺杆菌、排便异常、消化不适
- 肺部管理：吸烟风险、肺结节随访提醒、呼吸系统日常管理

每个专题都可以形成独立的健康档案，包括：

- 风险分层
- 指标趋势记录
- 生活方式建议
- 复查提醒
- 重点观察项

### 3. 个性化风险提醒

系统基于体检结果、症状问卷、生活习惯和既往记录，对用户进行持续风险提醒，例如：

- 某项指标连续升高，提示复查
- 某器官风险处于上升趋势，建议尽快干预
- 某种生活习惯可能加重当前风险
- 某类症状与既有体检结果结合后值得进一步关注

风险提醒的目标不是制造焦虑，而是帮助用户更早采取行动。

### 4. 生活方式建议生成

针对不同器官系统，VitaAI 会生成更具针对性的生活方式建议，而不是统一模板式建议。

例如：

- 肝脏方向：饮酒控制、作息调整、体重管理、控糖控脂建议
- 心血管方向：盐摄入管理、有氧运动建议、睡眠管理、压力管理
- 肠胃方向：饮食节律、刺激性食物规避、进食习惯优化
- 肺部方向：戒烟支持、空气暴露管理、呼吸训练与运动建议

建议会尽量做到：

- 与用户当前风险相关
- 可理解
- 可执行
- 可持续追踪

## 用户使用流程

普通用户可以按照以下方式使用 VitaAI：

1. 上传体检报告、化验单或输入关键健康数据
2. 填写基础问卷，包括年龄、性别、症状、生活习惯、家族史等
3. 系统自动识别异常指标并生成解读结果
4. 将风险归类到对应器官专题中
5. 生成个性化风险提醒和生活方式建议
6. 根据建议进行日常管理，并在后续体检或复查时持续更新

## 典型使用场景

### 场景一：体检后看不懂报告

用户拿到体检报告后，系统自动标记异常项，并解释这些异常与哪些器官风险相关，帮助用户判断哪些问题需要尽快重视。

### 场景二：某个器官有长期担忧

例如用户长期担心脂肪肝、心血管风险或胃部不适，系统可以围绕该器官建立长期管理档案，而不是只做一次性解读。

### 场景三：想知道接下来该做什么

很多人知道自己“有点异常”，但不知道下一步该控制饮食、增加运动、减少熬夜，还是尽快复查。VitaAI 的重点是把“看报告”转化为“下一步行动建议”。

## 系统价值

VitaAI 希望解决的不是“有没有病”的最终判断，而是健康管理中的几个高频问题：

- 体检结果信息量大，但普通用户理解门槛高
- 健康风险往往是长期积累，不是一次诊断就能解决
- 用户更需要持续提醒和具体行动，而不是一次性结论
- 不同器官问题背后往往与饮食、睡眠、运动、压力等生活方式高度相关

## 产品边界

为了避免误导，VitaAI 明确不做以下事情：

- 不替代医生诊断
- 不替代线下检查、影像学判断或专科就诊
- 不直接输出确定性的疾病结论
- 不在紧急情况中充当急救建议工具

以下情况应优先线下就医，而不是依赖系统建议：

- 持续胸痛、呼吸困难、晕厥等急症表现
- 明显消化道出血、剧烈腹痛、高热等情况
- 报告中出现需要医生进一步明确的重大异常
- 医生已经建议进一步检查或治疗的情况

## 未来方向

VitaAI 后续可以继续扩展：

- 对接可穿戴设备与连续健康数据
- 增加复查计划和随访机制
- 引入多模态健康档案，包括影像、报告、问卷和行为数据
- 构建面向家庭成员的联合健康管理能力
- 与医生、健康管理师或体检机构形成协同服务闭环

## 愿景

我们希望 VitaAI 能成为用户的长期健康助手：

- 帮助用户更早理解风险
- 帮助用户围绕重点器官做长期预防
- 帮助用户把体检结果变成真正可执行的健康行动

不是在疾病出现后才被动应对，而是在风险形成过程中尽早管理。
